# User Story: EGO Detail Page

## Description
As a user, I want to see the detailed info of EGOs so that I can determine what to add to build a concept deck.

## Acceptance Criteria
- The overview is the same as the identity counterpart
- Drop the sanity section
- Add sanity cost after the attack weight
- Revise the status section to show the sin cost and sin resistance
