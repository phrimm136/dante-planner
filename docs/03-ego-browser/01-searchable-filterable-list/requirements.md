# User Story: Searchable & Filterable List

## Description
As a user, I want to browse, search, and filter EGOs so that I can find desirable EGOs building a deck.

## Acceptance Criteria
- When users go to the `/ego` page, all EGOs are in the list.
- On the list, there are two selection bar, one for a filter on the left, another for a search bar on the right.
- The filter is consisted of the keywords (burn, bleed, tremor, rupture, sinking, poise, charge)
- The search bar supports international query; can find EGOs with korean words
- Clicking an EGO links to the details page
