# Task: Implement Global Layout Component

## Description
Build and integrate the GlobalLayout component using Tailwind and shadcn styling. The layout have to include header, body, and footer sections, which are always shown in every route. Note that the color of header and footer is differentiated from the global background color, while that of body is not.

## Research
- See the global theme

## Scope
- `frontend/src/styles/globals.css`

## Target Code Areas
- `front/src/components/`

## Testing Guidelines
- The header always have to be on the top of the page.
- The footer always have to be on the bottom of the page.
- The body always have to be between the header and footer and include all contents that will be filled later.
- The background color of header and footer is same for each other, but different from that of the body and the global background color.
