# User Story: Implement Header Components

## Description
As a user, I want to see the website title linked to the root page, navigator, and site-wise setting buttons (languages, settings, light/dark mode, login/logout) so that I can access them quickly.

## Acceptance Criteria
- The title is the leftmost linked to the root page.
- The navigators are next to the title name, each of which links to the corresponding page.
- The site-wise setting buttons are the rightmost, clicking them conducts their own tasks.
