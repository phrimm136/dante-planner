COLOR KEY (for identification only - use global theme for actual styling):
Top-Left Quadrant (Header):
- YELLOW: Grade icon (top-left corner)
- ORANGE: Identity name (below grade)
- RED: Character portrait image (large center)
- BLACK: Image toggle icon (on portrait)
- GRAY: Full-size icon (on portrait)
- GREEN: Three horizontal status panel containers (below portrait)

Top-Right Quadrant (Skills):
- BLUE: Skills panel outer container
- PURPLE: Skill selector row with clickable buttons for Skill 1, Skill 2, Skill 3, Defense
- RED: Skill image composite area
- ORANGE: Skill name with pill background
- YELLOW: Base power display
- GREEN: Attack weight display
- CYAN: Skill description container
- BLUE: Base description section
- MAGENTA: Coin number icons
- PINK: Coin effect text sections

Bottom-Left Quadrant (Sanity):
- BLUE: Sanity panel outer container
- RED: Panic type icon area (with RG/GRN sub-boxes)
- BLUE: Panic type text
- ORANGE: Sanity increment row (with GN sub-box)
- BLUE: Sanity increment text
- YELLOW: Sanity decrement row (with YW sub-box)
- BLUE: Sanity decrement text

Bottom-Right Quadrant (Passives):
- BLUE: Passives panel outer container
- PURPLE: Section headers (Passive, Support Passive)
- RED: Passive name with pill background
- ORANGE: Activation condition
- YELLOW: Passive effect description
- BLACK: Divider between multiple passives

LAYOUT NOTES:
- Status panels are THREE HORIZONTAL panels in a row (not vertical stack)
- Skills panel HAS skill selector (purple) at top with 4 buttons: Skill 1, Skill 2, Skill 3, Defense
- Active skill button is highlighted, default to Skill 1
- Skill content below selector switches when different button clicked
- Skill info row: Name (orange pill) + Base Power (yellow) + Attack Weight (green) horizontally
- Sanity panel has THREE rows vertically: Panic Type, Increment, Decrement
- Each sanity row has small colored icon box on left + label/description on right
- Passives can stack multiple entries vertically within each section
