# User Story: ID Detail Page

## Description
As a user, I want to see identity info, images, skill/passive texts so that I can find this identity's performance, which is needed for deck building

## Acceptance Criteria
- The page must show status, skills, passives, and sanity
- The description style must follow that of the game
- Use in-game images, sprites and layout for ease of identification
- Color (FG, BG): Wrath (#fe0000, #fe0000), Lust (#f86300, #fe4000), Sloth (#f4c528, #fefe00), Gluttony (#9dfe00, #40fe00), Gloom (#0dc1eb, #00fefe), Pride (#0048cc, #0040fe), Envy (#9300db, #fe00fe), def (#9f693a, #e9c99f)
