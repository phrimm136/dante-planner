# Dante's Planner

## Project Overview

A community-driven web application enabling Limbus Company players to create, share, and discuss game strategies, mainly for Mirror Dungeon. Users can find information about Identities/EGOs/EGO gifts, build team compositions, plan gift/theme pack selections, track the current run, and share comprehensive guides with the community.

## User Journey

> A player uses the web app to plan, track, and share Mirror Dungeon runs.
1. Visit Homepage
  - User lands on the homepage of the website.
2. Discover & Browse Information
  - The user looks up Identities, EGOs, and Gifts to understand their stats and effects.
3. Plan the Mirror Dungeon Run
  - The user builds a team plan by assigning Identities and EGOs to a deck, choosing Graces and EGO Gifts search before the run, selecting Gifts and Theme Packs to be chosen during the run, and adding memos for each components.
4. Track Progress During the Run
  - The user uses the planner to update acquired Gifts and chosen Theme Packs, adjust skills EA, and edit the given deck.
5. Share. Discuss, and Import Plans
  - The user uploads their planner data as a post to a public board and discusses it with others. Users can import the uploaded data so that they can use the data to track progress.

## Requirements
- Only signed in users can upload discuss the plans. Guests/signed in users can import them into their local db.
- The website itself have to provide multi-language support - except for user-created contents.
