# Searchable & Filterable List

## Description
As a user, I want to see the list where I can search and filter identities so that I can find desirable ones.

## Acceptance Criteria
- When users go to the `/identity` page, all identities are in the list.
- On the list, there are two selection bar, one for a filter on the left, another for a search bar on the right.
- The filter is consisted of the keywords (burn, bleed, tremor, rupture, sinking, poise, charge)
- The search bar supports international query; can find identities with korean words
- Clicking an identity links to the details page
