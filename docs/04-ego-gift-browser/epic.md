# Epic: EGO Gift Browser

## Description
Users can browse and view detailed information about EGO gifts.
This feature supports the discovery phase of the Mirror Dungeon journey, helping users analyze stats and synergy potential before building their team.

## Acceptance Criteria
- Users can search EGO gifts by name, keyword, or traits.
- Clicking an gift opens a detail page showing descriptions, costs, keywords, and way to obtain.
- All data (image, JSON) is fetched from static R2 storage or local mock in dev phase.
- Data is cached aggressively to reduce network load.
