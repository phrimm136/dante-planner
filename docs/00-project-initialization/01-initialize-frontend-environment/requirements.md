# User Story: Initialize Frontend Environment

## Description
As a frontend developer, I want to scaffold the React + Vite + Tailwind + shadcn UI project configured with TanStack Query/Router and Vitest, so that I can develop features consistently.

## Acceptance Criteria
- A single root project directory exists, containing:
  - /frontend directory for all client-side code.
  - /backend directory for all server-side code.
  - Shared directories or files (e.g., /config, /docs, .env, README.md) are accessible to both.
