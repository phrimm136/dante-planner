# Component i18n Patterns

## Mandatory Rule

**ALL user-visible text MUST use `t()`** - No hardcoded strings in UI

## Forbidden Pattern

| Forbidden | Use Instead |
|-----------|-------------|
| `<span>Submit</span>` | `<span>{t('buttons.submit')}</span>` |
| `placeholder="Enter name"` | `placeholder={t('inputs.name')}` |
| `title="Delete"` | `title={t('actions.delete')}` |

## Micro-Suspense Pattern

Wrap only text, keep structure visible:

```typescript
// GOOD: Structure visible, text suspends
<div className="card">
  <Suspense fallback="">
    {t('card.title')}
  </Suspense>
</div>

// BAD: Entire card suspends
<Suspense fallback={<CardSkeleton />}>
  <div className="card">{t('card.title')}</div>
</Suspense>
```

**Reference:** `PassiveI18n.tsx`, `SkillInfoPanel.tsx`, `StyledName.tsx`
